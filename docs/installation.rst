Installation
==========
The package is available to download on pypi.

**Requirements:**

- Python **>= 3.9**

- CPU architecture: **x86_64** (supported: `windows-x86_64`, `macos-x86_64`, `linux-x86_64`)

.. tab:: Windows

   .. code-block:: powershell

      > pip install -U zyntex

.. tab:: MacOS / Linux

   .. code-block:: sh

      $ pip install -U zyntex
